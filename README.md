# -GTACTW-Port-RG40XX-MuOS
Public release of the GTA: Chinatown Wars port for MuOS on H700 Devices. Original port made for the R36S by Mafradon, all credits go to him, i only recompiled it to run on H700 and MuOS.

GTA: Chinatown Wars

Port of the Android version of GTA: Chinatown Wars for PortMaster-compatible devices.

Requirements

You must provide your own legally obtained copy of:

gtactw.apk
main.4.com.rockstargames.gtactw.obb
Installation
Open the gtactw folder.
Copy the following files into the folder:
gtactw.apk
main.4.com.rockstargames.gtactw.obb
Launch the port.

During the first launch, the installer will automatically:

Extract the game data from the OBB file.
Extract required assets from the APK.
Remove the APK and OBB after installation to save storage space.

The installation process may take several minutes. Do not power off the device during installation.

Controls

Uses the default PortMaster gamepad layout.

Exit Game

Press:

START + SELECT

Compatibility

Tested on:

Anbernic RG40XX H
muOS

Other H700-based devices may also work.
